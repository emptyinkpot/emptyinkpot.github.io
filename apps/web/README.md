# emptyinkpot / apps-web

`apps/web/` 是当前仓库唯一对外发布的 Astro 站点应用。

## 入口

- 站点源码：`apps/web/src/`
- 内容集合：`apps/web/src/content/`
- 共享样式：`apps/web/src/styles/global.css`
- 站点配置：`apps/web/astro.config.mjs`

## 常用命令

```bash
npm install
npm run dev
npm run build
npm run preview
```

如果从仓库根目录运行，也可以直接使用：

```bash
npm run dev
npm run build
npm run preview
```
