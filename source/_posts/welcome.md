这篇文章是博客的第一篇内容，用来确认站点已经成功初始化。

这个站点现在已经具备这些基础能力：

- 本地运行 Hexo 预览
- 推送到 GitHub 后自动发布到 GitHub Pages
- 生成归档页和 RSS
- 支持后续继续扩展主题与页面

## 下一步可以做什么

你接下来可以从这几个地方开始：

```bash
npm run server
```

本地预览博客；

```bash
npm run new "我的第一篇文章"
```

创建新文章；

修改 `_config.yml` 中的标题、描述、作者等内容，让站点更像你自己。

## 写作位置

所有文章都放在 `source/_posts/` 目录下，使用 Markdown 编写即可。

以后你每次提交到 `main`，GitHub Actions 都会自动重新构建站点。
