# Runtime Identity

MyBlog is a public projection shell over a living knowledge runtime.

The site identity is not "blog posts in categories". The site identity is a mixed-object surface where articles, notes, books, visuals, projects, GitHub objects, music and collections remain available in one continuous environment.

## Runtime Principles

- Everything stays alive.
- Opening context must not destroy the feed.
- Reading happens in a drawer or reader surface, not in a card wall.
- Collections organize attention; they do not own content existence.
- Runtime state is explicit and bounded.
- OpenList is file access and cold/blob storage, not CMS.
- Obsidian is authoring truth, not the public site directly.
